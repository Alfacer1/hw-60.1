# Express Server - Phase 1

Цей проект є базовою реалізацією сервера на Node.js з використанням фреймворку Express. Проект побудований за архітектурою MVC (Model-View-Controller) для забезпечення модульності та масштабованості коду.

## Технології
- **Runtime:** Node.js
- **Framework:** Express.js
- **Package Manager:** npm / Yarn
- **Pattern:** MVC

## Інструкції з встановлення та запуску

1. **Клонуйте репозиторій:**
   ```bash
   git clone <посилання-на-ваш-репозиторій>
   cd my-express-server

2. **Встановіть залежності**: npm install
3. **Запустіть сервер**: npm start
4. **Сервер почне прослуховування на порту 3000**.
